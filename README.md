- Requisitos:
  - Docker/Compose
  - node v15.x~
  - yarn v1.22^
- Instalando as dependencias

  - `yarn`

- Construindo os containers
  - `yarn docker:start`
- Criando as tabelas no banco de dados

  - `yarn db:push`

- A aplicacao estara rodando na porta 3000
